Dino - Match Making Game

  • Github Project

    https://github.com/leckchess/Dinoo

  1. Game Mechanics

    • The board features at least 5 different colored tiles and can vary in size (e.g., 5x5, 7x7, 6x9).
    • Adjacent tiles of the same color can be linked.
    • Links can be made horizontally, vertically, or diagonally.
    • A minimum of 3 tiles is required to make a match.
    • Matched tiles are removed from the board.
    • The board collapses downward to fill empty slots, and new tiles fall from the top.
    • If no valid links are available, the board reshuffles.
    • Links in progress can be undone by swiping back to the previous tile.

  2. Unity Editor Level Creator

    • Provide a level creator tool in Unity Editor for making updates without code changes.
